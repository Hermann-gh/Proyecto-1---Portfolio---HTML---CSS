# Proyecto 1 - HTML + CSS

## Descripción

Este proyecto consiste en el desarrollo de una aplicación web tipo portafolio personal realizada únicamente con **HTML5** y **CSS3**.

El objetivo principal es poner en práctica los conocimientos básicos de maquetación web, estructura semántica, diseño responsive y organización de estilos utilizando buenas prácticas de desarrollo frontend.

La aplicación incluye diferentes secciones como presentación personal, proyectos y formulario de contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Git & GitHub
